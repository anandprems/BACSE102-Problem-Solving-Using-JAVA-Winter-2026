## üìö Syllabus Modules (As per VIT University Curriculum - Winter 2026)

### **Module 1: Introduction to Java Fundamentals **  


OOP Paradigm - Features of Java Language - JVM - Bytecode - Java program structure  
Basic programming constructs - data types variables Java naming conventions operators  
Control and looping constructs - Arrays one dimensional and multi-dimensional  
enhanced for loop - Strings - StringBuffer, StringBuilder and Math - Wrapper classes.

---

### **Module 2: Object-Oriented Design: Classes, Inheritance, and Polymorphism **  

Class Fundamentals - Access and non-access specifiers - Declaring objects and assigning  
object reference variables ‚Äì array of objects - constructors and destructors - usage of  
"this" and "static" keywords. Enum Types and its iterations - Inheritance - types - use of  
"super" - final keyword - Polymorphism - Overloading and Overriding - abstract class - Interfaces.

---

### **Module 3: Packages, Exception Handling and Multithreading ** 

Packages Creating and Accessing - Sub packages - Exception Handling - Types of Exception  
- Control Flow in Exceptions - Use of try, catch, finally, throw, throws in Exception Handling  
- User defined exceptions. Basics of threads ¬∑ Defining threads with Runnable interface  
Multi Threads and Inter Thread Communications with Synchronization.

---

### **Module 4: IO Streams and Files **  

File Object - I/O streams ‚Äì FileInputStream & FileOutputStream - FileReader & FileWriter -  
DataInputStream & DataOutputStream ‚Äì BufferedInputStream & BufferedOutputStream ‚Äì  
PrintOutputStream - Serialization and Deserialization.

---

### **Module 5: Generic Programming and the Java Collections Framework **  

Generic classes and methods - Collection framework: List and Map - ArrayList, Linked List,  
TreeMap and HashMap, Lambda Function, Java Database Connectivity (JDBC), JDBC Architecture,  
Handling CRUD Operations with Databases (Create, Read, Update, Delete).
